# Pulse

A headless command engine with declarative Discord UI.

[**Read the full documentation in /docs/**]

## Features

- Unified command definition (metadata for all Discord methods)
- Declarative UI schema (fluent API and config)
- Automatic UI binding to logic (no manual handlers)
- Stateful UI sessions (persist, expire, clean-up)
- Command argument parser (typed, validated)
- Autocomplete for slash commands (async sources)
- Pagination API (integrated with UI/session)
- Adapter-powered (discord.js, py, ts)
- Introspection API (tree, docs, dashboards)
- Advanced error & UX handling
- Permission guard layer
